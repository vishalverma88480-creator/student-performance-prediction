# Student Performance Prediction

## Description
A machine learning project that predicts student final scores based on study hours and attendance.

## Technologies Used
- Python
- Pandas
- Scikit-learn

## Features
- Data preprocessing
- Model training using Linear Regression
- Performance evaluation

## How to Run
1. Install dependencies  
   pip install -r requirements.txt  
2. Run the script  
   python student_performance.py
